# Healthcare-risk-prediction-pipeline
## Author
Adrian Garnelo Fernandez
## Project Overview
This project builds a healthcare risk prediction pipeline using AWS cloud tools.  
The pipeline processes raw healthcare datasets, performs data cleaning, exploratory data analysis (EDA), KPI analysis, and prepares data for machine learning models.

## Tools
- AWS S3
- AWS Glue
- AWS Athena
- Python / PySpark
- SQL
- Machine Learning
